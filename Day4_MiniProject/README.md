# Array Analyzer (Java)

This is a simple Java console-based project called **Array Analyzer**.  
It helps analyze arrays using common DSA concepts like:

 1.Majority Element (Boyer-Moore Voting)
 2.Kadane’s Algorithm (Max Subarray Sum)
 3.Prefix Sum + HashMap (Subarray Sum K)
 4.Sliding Window (Longest Unique Substring)
 5.HashSet (Longest Consecutive Sequence, Find Duplicate)
 6.Frequency Counting (Map based)

This project is built to strengthen problem-solving skills and demonstrate how different DSA patterns can be applied to real-world array problems.



##  Features :

The program performs multiple analyses on an array:

1. **Majority Element (> n/2)**
2. **Majority Element II (> n/3)**
3. **Maximum Subarray Sum (Kadane’s Algorithm)**
4. **Longest Subarray with Sum = K**
5. **Longest Substring Without Repeating Characters**
6. **Longest Consecutive Sequence**
7. **Find Duplicate Number**
8. **Frequency Analysis of Array Elements**
